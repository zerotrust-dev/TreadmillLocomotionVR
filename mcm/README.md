# Treadmill Locomotion VR MCM

The native DLL exposes a small Papyrus API for the SkyUI VR MCM:

- `TreadmillLocomotionVR.psc` declares the native functions exported by the DLL.
- `TreadmillLocomotionVR_MCM.psc` builds the in-game menu.
- `TreadmillLocomotionVR_MCM.esp` is generated by `tools/McmPluginBuilder`.
- `Scripts/TreadmillLocomotionVR.pex` and
  `Scripts/TreadmillLocomotionVR_MCM.pex` are compiled in GitHub Actions.

The native DLL remains authoritative and writes
`Data\SKSE\Plugins\TreadmillLocomotionVR.ini`.
